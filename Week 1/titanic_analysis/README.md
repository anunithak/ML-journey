The goal of this mini project was to analyze survival patterns in the Titanic dataset
The dataset contains information about Titanic passengers, such as age, gender, ticket class, and survival status (1 = survived, 0 = did not survive).

Analysis:
Checked for missing data , there was mostly none
Added AgeGroup feature
Calculated survival rates by gender, class and agegroup
Created visualizations to highlight key patterns

Key Findings:
Women had a higher survival rate than men
passengers in the first class had a higher chance of survival
